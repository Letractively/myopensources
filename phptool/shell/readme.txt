--------------------
Introduction
--------------------
Welcome to database migration tool
This tool migrate data and table struct from one database to another
eg: from oracle to mysql. It implemented by php, run on console, easy
to use. 

--------------------
Install
--------------------
1. Extract files to a folder
2. Edit Application/Config.php for setting source and target database servers.
3. Run migratedb on console

--------------------
 Usage
--------------------

For linux/unix
~~~~~~~~~~~~~~
run migratedb

For windows
~~~~~~~~~~~~~~
run migratedb.bat
